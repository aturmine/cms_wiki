// Information sur le système

CMS_wiki est basé sur le système CPhalcon, une extension PHP fondé sur le langage C.

Base de donnée : MySQL
Version php : PHP 5.4
Service javascript : Bootstrap, jquery
Service Socket.io (node.js) : Non

// Nomenclature pour les feuilles de style

Styles reliés aux titres

h1 -> Titre d'articles
h2 -> Titre important dans une page avec section
h3 -> Titre de section dans une page
h4 -> Soustitre d'articles

Style reliés aux blocks des pages

#menu-h -> menu distinct des pages (header)
#menu-f -> menu distinct des pages (footer)

#menu-t -> menu en haut
#menu-r -> menu à droite

#page-l -> get-content() à gauche

Articles

.ariane-t -> Fils d'ariane en haut
